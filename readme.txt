== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rank-with-schema` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the "Rank With Schema" settings screen to configure the plugin

== Frequently Asked Questions ==

= How to install and use this Schema plugin? =

After you Active this plugin, just go to Dashboard > Rank With Schema Settings, and then setup the default settings, after that, just go back to Post/Page and add any data that you like!
